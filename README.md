# DT3-60FPS

A community project attempting to rewrite DT3's code to make it a 60FPS game.

This uses GameMaker 8.2, available here.

https://gm82.cherry-treehouse.com/

Super Sound Plugin also needed, available in discord.

1) Install GM8.2
2) Install Super Sound plugin by dragging it into GM8.2's folder of extensions.
3) Run DT3_v1.5.2.4.gm82
4) Install any missing fonts
