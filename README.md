# DT3-60FPS

!!!This repo has been deprecated!!! Look at the other one: https://github.com/Quof/DT3-60FPS-plus

A community project attempting to rewrite DT3's code to make it a 60FPS game.

This uses GameMaker 8.2, available below. 8.2 natively splits GameMaker project files into separate files to allow for source control, so no need to recombine files or anything like that.

https://gm82.cherry-treehouse.com/

Super Sound Plugin also needed, available in discord.

1) Install GM8.2; make sure your version is from 2026 or later
2) Install Super Sound plugin by dragging it into GM8.2's folder of extensions.
3) Run DT3_v1.5.2.4.gm82
4) Install GM8.2 Core in Extensions of GM8.2. You have to press the + and install it.
5) Install any missing fonts
